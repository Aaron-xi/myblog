<h2 class="wp-block-heading" id="toc-2-3">🏠 CasaOS 部署指南</h2>

<!-- /wp:heading -->



<!-- wp:paragraph -->

<p>如果你正在使用 CasaOS，那么部署 LibreTV 将会非常简单：</p>

<!-- /wp:paragraph -->



<!-- wp:list {"ordered":true} -->

<ol class="wp-block-list"><!-- wp:list-item -->

<li><strong>安装自定义应用：</strong> 在 CasaOS 的管理界面，点击右侧的 <strong>“+”</strong> 号，然后选择 <strong>“安装自定义应用”</strong>。</li>

<!-- /wp:list-item -->



<!-- wp:list-item -->

<li><strong>导入 Compose 文件：</strong> 进入自定义应用安装页面后，点击右上角的 <strong>“导入”</strong> 按钮。将 LibreTV 的 Docker Compose 文件内容粘贴到出现的文本框中。（<strong>Compose 文件代码我会在文章末尾提供</strong>）。</li>

<!-- /wp:list-item -->



<!-- wp:list-item -->

<li><strong>设置密码：</strong> 找到 Compose 文件中的 <code>PASSWORD</code> 这一行，将后面的默认值修改为你自己设定的密码。请务必设置一个安全的密码。</li>

<!-- /wp:list-item -->



<!-- wp:list-item -->

<li><strong>提交配置：</strong> 修改密码完成后，点击 <strong>“提交”</strong>。</li>

<!-- /wp:list-item -->



<!-- wp:list-item -->

<li><strong>设置 Web UI 端口：</strong> 在 <strong>“WEB UI”</strong> 这一栏，填写你希望 LibreTV 使用的端口号。你可以根据自己的网络情况修改此端口，避免冲突。</li>

<!-- /wp:list-item -->



<!-- wp:list-item -->

<li><strong>开始安装：</strong> 填写完毕后，点击 <strong>“安装”</strong>。</li>

<!-- /wp:list-item --></ol>

<!-- /wp:list -->



<!-- wp:paragraph -->

<p>安装完成后，在 CasaOS 的应用列表中找到 LibreTV 的图标并打开。首次打开会要求你输入之前设置的密码，输入后即可进入 LibreTV 的主界面。</p>

<!-- /wp:paragraph -->



<!-- wp:heading {"level":3} -->

<h3 class="wp-block-heading" id="toc-3-4">配置 LibreTV (CasaOS)</h3>

<!-- /wp:heading -->



<!-- wp:paragraph -->

<p>进入 LibreTV 后，点击右上角的&nbsp;<strong>设置图标</strong>。在这里，你会看到默认只选择了两个搜索源。为了获得更全面的搜索结果，<strong>强烈建议你勾选所有可用的搜索源</strong>。下方还有&nbsp;<strong>“自定义 API”</strong>&nbsp;选项，如果你有其他优质的影视资源站点并提供 API 接口，可以将其添加到这里扩展搜索能力。开启&nbsp;<strong>“豆瓣热门推荐”</strong>&nbsp;后，首页会显示豆瓣的热门影视资源，方便你发现感兴趣的内容。现在，你就可以直接点击首页推荐的视频进行播放，或者在搜索框中输入关键词进行搜索。播放速度和清晰度都非常不错。</p>

<!-- /wp:paragraph -->



<!-- wp:heading -->

<h2 class="wp-block-heading" id="toc-2-5">☁️ Cloudflare Pages 部署指南</h2>

<!-- /wp:heading -->



<!-- wp:paragraph -->

<p>如果你希望通过 Cloudflare Pages 部署 LibreTV，请注意由于 Cloudflare 的子域名在中国大陆可能无法直接访问，你可能需要准备一个域名。</p>

<!-- /wp:paragraph -->



<!-- wp:list {"ordered":true} -->

<ol class="wp-block-list"><!-- wp:list-item -->

<li><strong>Fork 项目到 GitHub：</strong> 前往 LibreTV 的 GitHub 仓库 [<a href="https://github.com/LibreSpark/LibreTV" target="_blank" rel="noreferrer noopener">https://github.com/LibreSpark/LibreTV</a> ] 。 在 Fork 之前，请考虑给作者点个 Star ⭐ 以示支持。点击右上角的 <strong>“Fork”</strong>，选择你的 GitHub 账户并创建 Fork。</li>

<!-- /wp:list-item -->



<!-- wp:list-item -->

<li><strong>创建 Cloudflare Pages 项目：</strong> 登录你的 Cloudflare 账户，进入左侧的 <strong>“Workers 和 Pages”</strong> 页面，点击 <strong>“创建项目”</strong>，然后选择 <strong>“Pages”</strong>。</li>

<!-- /wp:list-item -->



<!-- wp:list-item -->

<li><strong>连接 GitHub 账户：</strong> 点击 <strong>“连接到 GitHub”</strong>，授权 Cloudflare 访问你的 GitHub 账户。找到你 Fork 的 LibreTV 项目，然后点击 <strong>“选择开始部署”</strong>。</li>

<!-- /wp:list-item -->



<!-- wp:list-item -->

<li><strong>配置环境变量：</strong> 在项目设置中，点击下方的 <strong>“环境变量”</strong>，然后点击 <strong>“添加变量”</strong>。在 <strong>“变量名称”</strong> 中填写 <code>PASSWORD</code>（注意大小写），在 <strong>“值”</strong> 中填写你想要设置的 LibreTV 访问密码。完成后，点击 <strong>“保存并部署”</strong>。等待部署完成。</li>

<!-- /wp:list-item -->



<!-- wp:list-item -->

<li><strong>添加自定义域名（可选）：</strong> 部署成功后，选择 <strong>“继续处理项目”</strong>，然后选择 <strong>“自定义域”</strong>。在这里添加你自己的域名，点击 <strong>“继续”</strong>，然后点击 <strong>“激活域”</strong>。等待 DNS 解析生效并显示为 “活动” 状态后，你就可以通过你的域名访问 LibreTV 了。</li>

<!-- /wp:list-item --></ol>

<!-- /wp:list -->



<!-- wp:paragraph -->

<p>部署完成后，你就可以通过 Cloudflare Pages 提供的&nbsp;<code>.workers.dev</code>&nbsp;子域名（或你的自定义域名）访问 LibreTV，使用你设置的密码登录，配置和使用步骤与 CasaOS 上的版本完全一致。</p>
